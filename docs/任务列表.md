## 第三阶段：MCP 工具实现

### 3.1 工具定义
- [x] 实现 transcribe_video 工具（tools/transcribe.py）
  - 输入 URL，内部串联：下载视频 → 提取音频 → 上传OSS → 语音识别 → 生成SRT
  - 返回 task_id + 转录文本 + SRT 结构化数据
- [x] 实现 analyze_video_frames 工具（tools/analyze.py）
  - 输入 timestamps + task_id/video_path/url
  - 内部串联：获取视频 → 提取帧 → 上传OSS → 图像识别
  - 视频来源优先级：task_id > video_path > url
