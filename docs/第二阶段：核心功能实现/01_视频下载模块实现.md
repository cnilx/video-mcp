# 视频下载模块实现

## 完成时间
2026-03-10

## 支持的平台

- **YouTube** (youtube.com, youtu.be) - 支持多种质量选项
- **Bilibili** (bilibili.com, b23.tv) - 支持多种质量选项
- **抖音** (douyin.com, v.douyin.com) - 自动解析短链，提取视频标题

## 核心功能

### VideoDownloader 类

**文件**: `src/core/downloader.py`

主要功能：
- 异步下载支持，不阻塞主线程
- 多平台自动识别
- 可配置的下载参数（输出目录、文件大小限制、重试次数、超时时间）
- 下载进度实时跟踪
- 自动重试机制
- Cookie 支持（可选）
- Windows 文件名兼容

### 数据类

**VideoQuality** - 视频质量枚举
- BEST: 最佳质量
- HIGH: 高清（1080p）
- MEDIUM: 标清（720p）
- LOW: 低清（480p）

**DownloadProgress** - 下载进度信息
- 实时状态跟踪
- 下载速度和预计剩余时间
- 下载百分比计算

**DownloadResult** - 下载结果
- 成功/失败状态
- 文件信息（路径、标题、时长、大小、格式）
- 错误信息

### 核心方法

- `download()`: 异步下载视频
- `get_video_info()`: 获取视频信息（不下载）
- `detect_platform()`: 检测视频平台
- `set_progress_callback()`: 设置进度回调函数

## 使用示例

```python
import asyncio
from src.core.downloader import VideoDownloader, VideoQuality

async def main():
    downloader = VideoDownloader(
        output_dir="./downloads",
        max_file_size=100 * 1024 * 1024,  # 100MB
        max_retries=3
    )

    result = await downloader.download(
        url="https://www.youtube.com/watch?v=xxxxx",
        quality=VideoQuality.HIGH
    )

    if result.success:
        print(f"下载成功: {result.file_path}")
    else:
        print(f"下载失败: {result.error}")

asyncio.run(main())
```

## 技术特点

### 异步设计
- 使用 `asyncio` 实现异步下载
- 通过 `run_in_executor` 在线程池中执行 yt-dlp
- 支持并发下载多个视频

### 错误处理
- 完善的异常捕获和处理
- 自动重试机制（可配置重试次数）
- 详细的错误信息返回

### 进度跟踪
- 实时下载进度回调
- 支持自定义进度处理函数
- 提供下载速度和剩余时间估算
