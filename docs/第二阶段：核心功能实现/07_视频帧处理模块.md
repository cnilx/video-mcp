# 视频帧处理模块

## 概述

视频帧处理模块（`src/core/frames.py`）负责根据给定的时间戳列表从视频中提取帧图片，支持批量并发提取、格式/质量/缩放配置，以及自动上传到阿里云 OSS。

## 模块位置

```
src/core/frames.py
```

## 数据模型

### ImageFormat

图片格式枚举：

| 值 | 说明 |
|---|---|
| `PNG` | 无损格式 |
| `JPEG` | 有损压缩，默认格式 |
| `WEBP` | 现代格式，体积更小 |

### FrameConfig

帧提取配置：

| 字段 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `image_format` | `ImageFormat` | `JPEG` | 输出图片格式 |
| `quality` | `int` | `85` | JPEG/WebP 质量 1-100 |
| `max_width` | `Optional[int]` | `None` | 最大宽度（等比缩放） |
| `max_height` | `Optional[int]` | `None` | 最大高度（等比缩放） |
| `max_concurrent` | `int` | `4` | 并发提取数 |

### FrameInfo

单帧信息：

| 字段 | 类型 | 说明 |
|---|---|---|
| `index` | `int` | 帧序号 |
| `timestamp` | `float` | 时间戳（秒） |
| `file_path` | `str` | 本地文件路径 |
| `oss_url` | `str` | OSS 公网 URL |
| `width` | `int` | 图片宽度 |
| `height` | `int` | 图片高度 |
| `file_size` | `int` | 文件大小（字节） |

### ExtractionResult

提取结果：

| 字段 | 类型 | 说明 |
|---|---|---|
| `success` | `bool` | 是否成功 |
| `frames` | `List[FrameInfo]` | 帧信息列表 |
| `total_requested` | `int` | 请求提取的帧数 |
| `total_extracted` | `int` | 实际提取成功的帧数 |
| `video_duration` | `float` | 视频总时长（秒） |
| `error` | `Optional[str]` | 错误信息 |

## 核心类 FrameExtractor

### 初始化

```python
from src.core.frames import FrameExtractor, FrameConfig, ImageFormat
from src.utils.oss import OSSUploader

# 默认配置
extractor = FrameExtractor()

# 自定义配置
config = FrameConfig(
    image_format=ImageFormat.JPEG,
    quality=90,
    max_width=1280,
    max_concurrent=4,
)
oss = OSSUploader()
extractor = FrameExtractor(config=config, oss_uploader=oss)
```

### 公开方法

#### `extract_frames(video_path, output_dir, timestamps) -> ExtractionResult`

根据时间戳列表提取帧，可选上传 OSS。

```python
result = await extractor.extract_frames(
    video_path="downloads/video.mp4",
    output_dir="workspace/frames",
    timestamps=[5.0, 30.0, 60.0, 120.5],
)

if result.success:
    for frame in result.frames:
        print(f"帧 {frame.index}: {frame.timestamp}s -> {frame.oss_url}")
```

## 实现细节

### ffmpeg 调用

使用 `ffmpeg-python` 库，通过 input seeking（`ss` 参数在 input 上）实现快速定位：

```python
ffmpeg.input(video_path, ss=timestamp).output(output_path, vframes=1)
```

### 并发控制

使用 `asyncio.Semaphore` 限制并发数（默认 4），通过 `asyncio.gather` 并行提取：

```python
semaphore = asyncio.Semaphore(config.max_concurrent)
```

### 缩放

可选的 ffmpeg scale 滤镜，保持原始宽高比：

```python
ffmpeg.filter(stream, 'scale', w, h, force_original_aspect_ratio='decrease')
```

### 时间戳校验

自动过滤超出视频时长的时间戳，并输出 warning 日志。

### OSS 上传

复用 `src/utils/oss.py` 的 `OSSUploader`，提取完成后批量上传到 `frames/` 目录。
