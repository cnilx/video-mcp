# 音频处理模块 - 最新实现

## 更新日期
2026-03-11

## 概述

音频处理模块负责从视频中提取音频、进行语音识别，并生成 SRT 字幕文件。

## 核心功能

### 1. 音频提取
- 使用 ffmpeg 从视频中提取音频
- 支持多种音频格式（MP3、WAV、FLAC 等）
- 自动调整采样率和声道数

### 2. 语音识别
- 使用阿里百炼 **qwen3-asr-flash-filetrans** 模型
- 支持任意长度音频（无需分片）
- 提供句子级毫秒精度时间戳
- 自动语言识别

### 3. SRT 字幕生成
- 自动生成标准 SRT 格式字幕
- 基于句子级时间戳
- 可直接用于视频编辑软件

### 4. OSS 集成
- 自动上传音频到阿里云 OSS
- 生成公网可访问 URL
- 支持 filetrans 模型的文件访问要求

## 技术架构

### 处理流程

```
本地音频文件
    ↓
上传到 OSS
    ↓
获取公网 URL
    ↓
提交 filetrans 转录任务
    ↓
等待任务完成（异步）
    ↓
下载转录结果
    ↓
解析句子和时间戳
    ↓
生成 SRT 字幕文件
```

### 核心类

#### AudioProcessor

主要的音频处理类，提供以下功能：

```python
class AudioProcessor:
    def __init__(
        self,
        api_key: Optional[str] = None,
        model: str = "qwen3-asr-flash-filetrans",
        language: str = "zh",
        oss_uploader: Optional[OSSUploader] = None,
    ):
        """初始化音频处理器"""

    async def extract_audio(
        self,
        video_path: str,
        output_path: Optional[str] = None,
        audio_format: AudioFormat = AudioFormat.MP3,
        sample_rate: Optional[SampleRate] = None,
        audio_bitrate: str = "128k",
    ) -> Optional[str]:
        """从视频中提取音频"""

    async def transcribe_audio(
        self,
        audio_path: str,
        use_filetrans: bool = True,
    ) -> TranscriptionResult:
        """语音转文本"""

    async def save_srt(
        self,
        result: TranscriptionResult,
        output_path: str,
    ) -> bool:
        """保存为 SRT 字幕文件"""
```

## 使用方法

### 基本用法

```python
from src.core.audio import AudioProcessor
from src.utils.oss import OSSUploader

# 初始化
oss_uploader = OSSUploader()
processor = AudioProcessor(
    api_key="your-dashscope-api-key",
    oss_uploader=oss_uploader,
)

# 提取音频
audio_path = await processor.extract_audio("video.mp4")

# 转录音频
result = await processor.transcribe_audio(audio_path)

if result.success:
    # 打印转录文本
    print(result.text)

    # 保存 SRT 字幕
    await processor.save_srt(result, "output.srt")
```

### 使用 DashScope SDK

实际实现中使用 DashScope SDK 的方式：

```python
from dashscope.audio.qwen_asr import QwenTranscription
import dashscope

# 设置 API Key
dashscope.api_key = "your-api-key"
dashscope.base_http_api_url = 'https://dashscope.aliyuncs.com/api/v1'

# 上传音频到 OSS
oss = OSSUploader()
audio_url = oss.upload_file("audio.mp3")

# 提交转录任务
task_response = QwenTranscription.async_call(
    model='qwen3-asr-flash-filetrans',
    file_url=audio_url,
    enable_itn=True,
    enable_words=True,
)

# 等待任务完成
result = QwenTranscription.wait(task=task_response.output.task_id)

# 下载结果
import requests
transcription_url = result.output.result['transcription_url']
result_data = requests.get(transcription_url).json()

# 解析句子
sentences = result_data["transcripts"][0]["sentences"]
```

## 注意事项

1. **OSS 配置**: 使用 filetrans 模型必须配置 OSS
2. **Bucket 权限**: 建议设置为公共读或配置正确的访问权限
3. **文件清理**: 建议定期清理 OSS 中的临时音频文件
4. **网络要求**: 需要稳定的网络连接上传文件
5. **API 限制**: 注意 API 调用频率限制

## 故障排查

### 问题 1: OSS 上传失败

**可能原因**:
- OSS 配置错误
- 网络连接问题
- Bucket 权限不足

**解决方法**:
- 检查环境变量配置
- 验证 OSS AccessKey 是否正确
- 确认 Bucket 存在且有写入权限

### 问题 2: 转录任务失败

**可能原因**:
- 音频 URL 无法访问
- API Key 无效
- 音频格式不支持

**解决方法**:
- 确认 OSS URL 可以公开访问
- 验证 DASHSCOPE_API_KEY 是否正确
- 检查音频文件格式

### 问题 3: 分句效果不好

**解决方法**:
- 确保启用了 `enable_itn=True` 和 `enable_words=True`
- 检查音频质量
- 尝试调整语言设置
