# 全流程集成测试指南

## 概述

`test_full_workflow.py` 提供了完整的视频处理流程测试，包括：

1. **视频下载** - 从URL下载视频
2. **音频提取** - 从视频中提取音频
3. **语音识别** - 使用阿里百炼 Qwen-ASR 进行语音转文本
4. **生成字幕** - 生成SRT字幕文件

所有生成的文件都保存在 `./testWorkspace` 目录中。

## 环境配置

### 必需的环境变量

在项目根目录的 `.env` 文件中配置以下变量：

```bash
# 阿里百炼 API Key（必需）
DASHSCOPE_API_KEY=your_dashscope_api_key

# OSS 配置（语音识别需要）
OSS_ACCESS_KEY_ID=your_oss_access_key_id
OSS_ACCESS_KEY_SECRET=your_oss_access_key_secret
OSS_ENDPOINT=your_oss_endpoint
OSS_BUCKET_NAME=your_oss_bucket_name

# 测试视频URL（可选，用于在线下载测试）
TEST_VIDEO_URL=https://example.com/test_video.mp4

# 本地测试视频路径（可选，用于本地文件测试）
TEST_LOCAL_VIDEO=./downloads/test_video.mp4
```

### 依赖安装

确保已安装所有必需的依赖：

```bash
# 激活虚拟环境
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```

## 运行测试

### 方式1: 使用 pytest 运行

```bash
# 运行所有测试
pytest tests/test_full_workflow.py -v -s

# 运行特定测试
pytest tests/test_full_workflow.py::TestFullWorkflow::test_complete_workflow_from_url -v -s

# 运行本地视频测试
pytest tests/test_full_workflow.py::TestFullWorkflow::test_workflow_with_local_video -v -s
```

```bash
python tests/test_full_workflow.py
```

## 测试用例说明

### 1. test_complete_workflow_from_url

完整流程测试，从URL下载视频开始：

**前置条件：**
- 配置 `DASHSCOPE_API_KEY`
- 配置 OSS 相关环境变量
- 配置 `TEST_VIDEO_URL`

**测试步骤：**
1. 从URL下载视频到 `./testWorkspace`
2. 从视频中提取音频（MP3格式）
3. 使用 filetrans 模型进行语音识别

**输出文件：**
- `video_*.mp4` - 下载的视频文件
- `audio_*.mp3` - 提取的音频文件
- `transcription_*.txt` - 识别结果文本
- `subtitle_*.srt` - SRT字幕文件

### 2. test_workflow_with_local_video

使用本地视频文件的测试：

**前置条件：**
- 配置 `DASHSCOPE_API_KEY`
- 配置 OSS 相关环境变量
- 配置 `TEST_LOCAL_VIDEO` 并确保文件存在

**测试步骤：**
1. 从本地视频提取音频
2. 语音识别
3. 生成SRT字幕

**输出文件：**
- `audio_local_*.mp3` - 提取的音频文件
- `subtitle_local_*.srt` - SRT字幕文件

## 注意事项

### 1. 视频选择建议

- **测试视频时长**: 建议使用 1-5 分钟的短视频进行测试
- **视频质量**: 测试时使用 `MEDIUM` 质量以节省时间和带宽
- **音频清晰度**: 选择音频清晰的视频以获得更好的识别效果

### 2. 语音识别模型

测试使用 `qwen3-asr-flash-filetrans` 模型，特点：

- 支持长音频（无需手动分段）
- 提供句子级别的时间戳
- 需要 OSS 支持（音频文件需上传到 OSS）
- 识别准确率高

### 3. 费用说明

- **视频下载**: 免费（消耗网络流量）
- **OSS 存储**: 按阿里云 OSS 计费标准
- **语音识别**: 按阿里百炼 API 计费标准（按音频时长计费）

### 4. 常见问题

**Q: 测试跳过，提示未配置环境变量？**

A: 检查 `.env` 文件是否存在，环境变量是否正确配置。

**Q: OSS 上传失败？**

A: 检查 OSS 配置是否正确，Bucket 是否存在，权限是否足够。

**Q: 语音识别失败？**

A: 检查 API Key 是否有效，账户余额是否充足。

**Q: 视频下载失败？**

A: 检查视频 URL 是否有效，网络连接是否正常。
