# 抖音视频下载解决方案

## 问题描述

使用 yt-dlp 下载抖音视频时遇到 Cookie 认证错误：
```
ERROR: [Douyin] Fresh cookies (not necessarily logged in) are needed
```

## 问题原因

1. yt-dlp 自动识别抖音链接并切换到 Douyin 提取器
2. Douyin 提取器尝试调用抖音 API 获取视频信息
3. 抖音 API 需要 Cookie 认证才能访问
4. `force_generic_extractor` 配置在重定向后失效

## 解决方案

### 核心思路

从移动端页面提取视频播放地址，绕过 Douyin 提取器的 API 限制：

1. 使用移动端 User-Agent 访问抖音短链
2. 从页面 HTML 中提取 JSON 数据（`window._ROUTER_DATA`）
3. 从 JSON 中提取真实的视频播放地址和标题
4. 使用提取的播放地址直接下载视频

### 实现步骤

#### 1. 访问移动端页面

```python
headers = {
    'User-Agent': 'Mozilla/5.0 (iPhone; CPU iPhone OS 17_0 like Mac OS X) AppleWebKit/605.1.15 (KHTML, like Gecko) Mobile/15E148',
    'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8',
    'Accept-Language': 'zh-CN,zh;q=0.9',
}

async with httpx.AsyncClient(follow_redirects=True, timeout=30.0) as client:
    response = await client.get(url, headers=headers)
    html = response.text
```

#### 2. 提取 JSON 数据

```python
import re
import json

json_pattern = r'<script[^>]*>\s*window\._ROUTER_DATA\s*=\s*({.*?})\s*</script>'
matches = re.findall(json_pattern, html, re.DOTALL)
data = json.loads(matches[0])
```

#### 3. 提取视频信息

```python
item = data['loaderData']['video_(id)/page']['videoInfoRes']['item_list'][0]

# 提取播放地址
video_info = item['video']
play_addr = video_info['play_addr']['url_list'][0]

# 提取标题
title = item.get('desc', 'douyin_video')
title = re.sub(r'[<>:"/\\|?*]', '', title)  # 清理特殊字符
```

#### 4. 下载视频

```python
ydl_opts = {
    'outtmpl': f'./downloads/{title}.%(ext)s',
    'http_headers': {
        'User-Agent': 'Mozilla/5.0 (iPhone; CPU iPhone OS 17_0 like Mac OS X) AppleWebKit/605.1.15',
        'Referer': 'https://www.douyin.com/',
    },
}

with yt_dlp.YoutubeDL(ydl_opts) as ydl:
    ydl.download([play_addr])
```

## 技术细节

### 重定向流程

```
v.douyin.com (短链)
  ↓
iesdouyin.com (移动端页面)
  ↓
提取 window._ROUTER_DATA JSON
  ↓
aweme.snssdk.com (播放 API)
  ↓
douyinvod.com (CDN 直链)
  ↓
下载视频文件
```

### JSON 数据结构

```json
{
  "loaderData": {
    "video_(id)/page": {
      "videoInfoRes": {
        "item_list": [
          {
            "desc": "视频标题",
            "video": {
              "play_addr": {
                "url_list": [
                  "https://aweme.snssdk.com/aweme/v1/playwm/..."
                ]
              }
            }
          }
        ]
      }
    }
  }
}
```

## 关键优势

- ✅ **不需要 Cookie** - 无需登录抖音账号
- ✅ **不需要关闭浏览器** - 不依赖浏览器 Cookie
- ✅ **自动提取标题** - 文件名使用视频标题而非乱码
- ✅ **稳定可靠** - 直接使用 CDN 直链
- ✅ **速度快** - 绕过 API 限制

## 失败的尝试

以下方案都尝试过但失败了：

| 方案 | 结果 | 原因 |
|------|------|------|
| 强制 Generic 提取器 | ❌ | 重定向后失效 |
| 使用浏览器 Cookie | ❌ | 浏览器锁定数据库 |
| 手动解析桌面端页面 | ❌ | 页面不包含视频 URL |
| 增强 HTTP Headers | ❌ | 仍然需要 Cookie |

## 经验教训

1. **分析成功案例很重要** - 从成功案例中找到关键线索
2. **移动端往往更容易** - 移动端页面通常包含更多数据
3. **直链优于 API** - CDN 直链比 API 更稳定
4. **不要过度依赖工具** - 有时需要自己提取数据
