# 音频转录和 SRT 生成快速指南

## 前置准备

### 1. 安装依赖

```bash
# 激活虚拟环境
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows

# 安装新依赖
pip install oss2
```

### 2. 配置环境变量

在 `.env` 文件中添加：

```bash
# 阿里百炼 API Key
DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxx

# 阿里云 OSS 配置
OSS_ACCESS_KEY_ID=LTAI5xxxxxxxxxxxxx
OSS_ACCESS_KEY_SECRET=xxxxxxxxxxxxxxxxxxxxxx
OSS_ENDPOINT=oss-cn-hangzhou.aliyuncs.com
OSS_BUCKET_NAME=your-bucket-name
```

### 3. 创建 OSS Bucket

1. 登录阿里云控制台
2. 进入 OSS 服务
3. 创建 Bucket（建议选择与你所在地最近的区域）
4. 设置读写权限为"公共读"或"私有"（推荐私有）


### 方法 2：在代码中使用

```python
import asyncio
from src.core.audio import AudioProcessor
from src.utils.oss import OSSUploader

async def transcribe_and_generate_srt():
    # 1. 初始化 OSS 上传器
    oss_uploader = OSSUploader()

    # 2. 初始化音频处理器
    processor = AudioProcessor(
        model="qwen3-asr-flash-filetrans",
        oss_uploader=oss_uploader,
    )

    # 3. 转录音频
    result = await processor.transcribe_audio("your_audio.mp3")

    if result.success:
        # 4. 打印转录文本
        print("转录文本:")
        print(result.text)

        # 5. 生成 SRT 文件
        await processor.save_srt(result, "output.srt")
        print("SRT 文件已生成: output.srt")
    else:
        print(f"转录失败: {result.error}")

# 运行
asyncio.run(transcribe_and_generate_srt())
```

## 常见问题

### Q1: 如何处理长音频？

A: 新的 filetrans 模型支持任意长度的音频，无需分片。

```python
# 直接转录，无需担心长度
result = await processor.transcribe_audio("long_audio.mp3")
```

### Q2: 如何使用旧模型（不需要 OSS）？

A: 设置 `use_filetrans=False`：

```python
processor = AudioProcessor(model="qwen3-asr-flash")
result = await processor.transcribe_audio(
    "audio.mp3",
    use_filetrans=False,
    auto_split=True,  # 自动分片
)
```

### Q3: SRT 文件可以用在哪里？

A: SRT 是通用字幕格式，可用于：
- 视频编辑软件（Premiere、Final Cut Pro）
- 视频播放器（VLC、PotPlayer）
- 视频平台（YouTube、B站）
- 字幕编辑工具

### Q4: 如何调整时间戳精度？

A: filetrans 模型自动提供毫秒级精度，无需调整。

### Q5: 转录支持哪些语言？

A: 支持多种语言，通过 `language` 参数设置：

```python
processor = AudioProcessor(
    language="en",  # 英语
    # language="zh",  # 中文（默认）
    # language="ja",  # 日语
    # language="ko",  # 韩语
)
```

1. **及时清理 OSS 文件**：转录完成后删除临时文件
2. **使用私有 Bucket**：保护音频隐私

4. **错误处理**：添加重试机制处理网络问题
