# 配置管理

## 配置文件

### 配置文件位置

- 默认配置：`config/config.json`
- 可通过环境变量 `CONFIG_PATH` 指定自定义路径

### 配置文件结构

```json
{
  "server": {
    "host": "0.0.0.0",
    "port": 8000,
    "timeout": 3600
  },
  "speech": {
    "base_url": "https://dashscope.aliyuncs.com/compatible-mode/v1",
    "model": "qwen3-asr-flash",
    "enable_itn": false,
    "language": null
  },
  "vision": {
    "base_url": "https://dashscope.aliyuncs.com/compatible-mode/v1",
    "model": "qwen3-vl-flash",
    "max_tokens": 2000,
    "temperature": 0.7
  },
  "workspace": {
    "base_dir": "/data/workspaces",
    "auto_cleanup_days": 7
  },
  "download": {
    "default_quality": "best",
    "max_file_size_gb": 5
  }
}
```

## 环境变量

### 必需的环境变量

在项目根目录创建 `.env` 文件：

```bash
# API 服务认证（必需）
API_KEY=your-secure-api-key-here

# 阿里百炼 API Key（必需）
DASHSCOPE_API_KEY=your-dashscope-api-key
```

### 可选的环境变量

```bash
# 自定义配置文件路径
CONFIG_PATH=config/custom-config.json
```

## 配置项说明

### 服务器配置 (server)

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| host | string | "0.0.0.0" | 服务器监听地址 |
| port | int | 8000 | 服务器端口（1-65535） |
| timeout | int | 3600 | 请求超时时间（秒） |

### 语音识别配置 (speech)

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| base_url | string | - | API 基础 URL |
| model | string | "qwen3-asr-flash" | 语音识别模型 |
| enable_itn | bool | false | 是否启用逆文本归一化 |
| language | string | null | 语言代码（如 "zh", "en"） |

### 图像识别配置 (vision)

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| base_url | string | - | API 基础 URL |
| model | string | "qwen3-vl-flash" | 图像识别模型 |
| max_tokens | int | 2000 | 最大 token 数 |
| temperature | float | 0.7 | 温度参数（0.0-2.0） |

### 工作空间配置 (workspace)

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| base_dir | string | "/data/workspaces" | 工作目录基础路径 |
| auto_cleanup_days | int | 7 | 自动清理天数 |

### 下载配置 (download)

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| default_quality | string | "best" | 默认视频质量 |
| max_file_size_gb | int | 5 | 最大文件大小（GB） |

## 使用方法

### 基本使用

```python
from src.utils.config import config

# 获取配置项
host = config.server_host
port = config.server_port

# 使用 get 方法（支持点号路径）
timeout = config.get("server.timeout", 3600)

# 获取敏感信息
api_key = config.api_key
dashscope_key = config.dashscope_api_key
```

### 自定义配置路径

```python
from src.utils.config import Config

# 使用自定义配置文件
custom_config = Config(config_path="path/to/config.json")
```

### 配置热重载

```python
from src.utils.config import Config

# 启用自动重载
config = Config(auto_reload=True)

# 配置文件修改后会自动重新加载
host = config.server_host  # 自动检查并重载
```

### 手动重载配置

```python
from src.utils.config import config

# 手动重新加载配置
success = config.reload()
if success:
    print("配置重新加载成功")
```

## 配置验证

配置管理模块使用 Pydantic 进行配置验证。

### 验证规则

1. **端口号验证**：必须在 1-65535 之间
2. **视频质量验证**：只允许 best/worst/720p/1080p/480p
3. **数值范围验证**：
   - timeout >= 1
   - max_tokens >= 1
   - temperature: 0.0-2.0
   - auto_cleanup_days >= 1
   - max_file_size_gb >= 1

### 验证失败处理

- 配置文件格式错误或验证失败时，使用默认配置
- 所有错误和警告都会记录到日志中
- 缺少必需的环境变量时，记录警告信息

## 配置优先级

1. **环境变量**：敏感信息（API_KEY、DASHSCOPE_API_KEY）
2. **配置文件**：应用配置（config.json）
3. **默认值**：内置默认配置

## 最佳实践

### 1. 分离敏感信息

- 将 API Key 等敏感信息放在 `.env` 文件中
- 不要将 `.env` 文件提交到版本控制系统
- 使用 `.env.example` 作为模板

### 2. 配置热重载

- 开发环境可以启用 `auto_reload=True`
- 生产环境建议关闭自动重载，使用手动重载

### 3. 配置文件管理

- 为不同环境创建不同的配置文件
- 使用 `CONFIG_PATH` 环境变量切换配置
- 保持配置文件的 JSON 格式正确
