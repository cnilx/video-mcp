# 3.1 工具定义 - 实现文档

## 概述

实现了两个 MCP 工具，分别用于视频转录和视频帧分析。两个工具均串联已有的核心模块，提供端到端的处理能力。

## 文件结构

```
src/tools/
├── __init__.py          # 模块导出
├── transcribe.py        # 视频转录工具
└── analyze.py           # 视频帧分析工具
```

## transcribe_video 工具

**文件**: `src/tools/transcribe.py`
**类**: `TranscribeVideoTool`

### 处理流程

```
输入 URL → 创建工作空间 → 下载视频 → 提取音频 → 上传OSS → 语音识别 → 生成SRT → 返回结果
```

### 接口

```python
async def run(
    url: str,              # 视频 URL
    quality: str = "best", # 视频质量 (best/high/medium/low)
    language: str = "zh",  # 语言代码
) -> TranscribeResult
```

### 返回结构

| 字段 | 类型 | 说明 |
|------|------|------|
| success | bool | 是否成功 |
| task_id | str | 工作空间 ID，可用于后续帧分析 |
| text | str | 完整转录文本 |
| srt_content | str | SRT 字幕文件内容 |
| srt_entries | List[SRTEntry] | 结构化 SRT 条目列表 |
| duration | float | 音频时长（秒） |
| video_title | str | 视频标题 |
| error | str | 错误信息 |

## analyze_video_frames 工具

**文件**: `src/tools/analyze.py`
**类**: `AnalyzeVideoFramesTool`

### 处理流程

```
解析视频来源 → [可选下载] → 提取帧 → 上传OSS → 图像识别 → 返回结果
```

### 视频来源优先级

1. `task_id` - 复用已有工作空间中的视频（如 transcribe_video 下载的）
2. `video_path` - 使用本地视频文件
3. `url` - 从 URL 下载视频

### 接口

```python
async def run(
    timestamps: List[float],           # 时间戳列表（秒）
    task_id: Optional[str] = None,     # 已有任务 ID
    video_path: Optional[str] = None,  # 本地视频路径
    url: Optional[str] = None,         # 视频 URL
    analysis_type: str = "smart",      # 分析类型
    custom_prompt: Optional[str] = None,
) -> AnalyzeFramesResult
```

### 返回结构

| 字段 | 类型 | 说明 |
|------|------|------|
| success | bool | 是否成功 |
| task_id | str | 工作空间 ID |
| frames | List[FrameAnalysis] | 每帧分析结果 |
| total_requested | int | 请求帧数 |
| total_succeeded | int | 成功帧数 |
| error | str | 错误信息 |

### 分析类型

- `general` - 通用描述
- `detailed` - 详细描述
- `objects` - 物体识别
- `text` - 文字识别（OCR）
- `scene` - 场景识别
- `smart` - 智能识别（默认，自动判断内容类型）

## 典型使用场景

```python
# 场景 1: 先转录，再分析关键帧
transcribe_tool = TranscribeVideoTool()
result = await transcribe_tool.run(url="https://example.com/video.mp4")

# 使用同一个 task_id 分析帧，无需重新下载
analyze_tool = AnalyzeVideoFramesTool()
frames = await analyze_tool.run(
    timestamps=[10.0, 30.0, 60.0],
    task_id=result.task_id,
)

# 场景 2: 直接分析 URL 视频的帧
frames = await analyze_tool.run(
    timestamps=[5.0, 15.0],
    url="https://example.com/video.mp4",
    analysis_type="detailed",
)
```
